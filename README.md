# LeafTrack

LeafTrack is a Chrome extension that automatically tracks job applications using Google Sheets and Gmail integration.

## Features
- Save jobs from major job boards
- Google Sheets sync
- Gmail status updates
- Salary and location detection
- Timeline & dashboard analytics
- Resume tracking
- Dark mode
